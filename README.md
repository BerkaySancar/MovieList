# MovieList

It is an application where the movies that will be upcoming and now playing are listed and their details are shown.

## Features

- Swift UIKit
- Storyboard
- XIB.
- Auto Layout
- CocoaPods
- Alamofire, SDWebImage, SwiftLint.
- UITableView
- UICollectionView
- UIPageControl

## Architecture

- Protocol-Oriented MVVM.
